# Birthday & Event Timer

A Python countdown timer that calculates the remaining time for birthdays or special events in days or seconds.

## 🚀 Features
* Supports countdowns for custom events like birthdays.
* Allows users to choose between a daily or precise per-second countdown.
* Features a continuous live-updating console display.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Modules used:** `datetime`, `time`
